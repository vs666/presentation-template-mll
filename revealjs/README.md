# USING THIS REPOSITORY : 

```bash
$ npm init
$ npm start 
```

This will start a server on localhost-8000 by default. (to change, use --port [PORT NUMBER])

Then go to : localhost:8000/filename.html 

For customized themes, go to ./dist/theme and find various themes there that can be imported and used in the presentation's html file.

Other features include possibility of creating a md file filled with contents for presentation in it.
